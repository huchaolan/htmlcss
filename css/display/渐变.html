线性渐变

从上到下的线性渐变

.gradient-default {
    background:linear-gradient(orange,yellow);
}

从左到右线性渐变

.gradient-urd {
    background: linear-gradient(to rigth,orange,yellow);
}

使用角度的线性渐变

.gradient-angle {
    background:linear-gradient(135deg,orange,yellow);
}

指定多个等间距的色标

.gradient-colors {
    background: linear-gradient(to right,red,orange,yellow,white);
}

使用透明度

.gradient-opactity {
    background: linear-gradient(to right,rgba(0,0,255,0),rgba(0,255,0,1));
}

重复的线性渐变

.gradient-repeat {
    background: repeating-linear-gradient(-45deg,blue,blue 5px,white 5px,white 10px);
}
