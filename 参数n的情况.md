# 结构伪类选择器中的n是什么

## 接受伪类选择器n有4个

+ :nth-child(n)
+ :nth-last-child(n)
+ :nth-of-type(n)
+ :nth-last-of-type(n)

## n的七种形式

运算结果0为不选择元素。1,6,7为固定值，其它都是公式

1. 参数n为具体的数值
2. 参数n为表达式n*length
3. 参数n为表达式n*length+b
4. 参数n为表达式n+length(选择倒数length个元素)
5. 参数n为表达式-n+length(选择正数length个元素)
6. 参数n为关键词odd(选择偶数)
7. 参数n为关键词even(选择奇数)
